# API Calls Documentation

Este pacote contém exemplos de chamadas à API da aplicação, utilizando curl e scripts automatizados, para facilitar testes e integração sem Postman.

## Estrutura
- healthcheck.md: exemplo de chamada ao endpoint de healthcheck.
- healthcheck.bat: script para executar chamada curl automaticamente no Windows.
- Outros arquivos: exemplos para cada endpoint da API.

## Como usar
1. Execute o script healthcheck.bat para testar o endpoint /health.
2. Ou copie o comando curl do arquivo correspondente e execute no terminal.
3. Verifique a resposta conforme o exemplo.

Adicione novos exemplos e scripts conforme endpoints forem implementados.
