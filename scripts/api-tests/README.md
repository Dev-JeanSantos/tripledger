# API HTTP Client Scripts

Esta pasta contém scripts de teste para endpoints da API, compatíveis com o HTTP Client integrado do JetBrains (IntelliJ, etc).

## Como usar
1. Abra o arquivo .http no editor JetBrains.
2. Clique em 'Run' acima da requisição para executá-la e ver a resposta.

Inclua novos arquivos .http para cada endpoint a ser testado.
